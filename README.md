# Dutch_Rental_Hunter

<h2>House Rental Check Application</h2>
This is a full-stack web application built with React, Node.js, Express, and PostgreSQL that help user to do rental house hunting nearby Amsterdam city.

<h2>Technologies</h2>
<li>Postgres</li>
<li>Express</li>
<li>React</li>
<li>Node</li>
<li>Material UI</li>

<h2>Features</h2>
<li>View rental apartment which publish on today.</li>
<li>Select the period to view historical rental records</li>

<h2>Setup</h2>
Follow the instruction below to run locally.
<h3>Installation</h3>

Step 1. Clone the repository:
```
git clone git@github.com:yuch3628/Dutch_Rental_Hunter.git
```
Step 2. Install the dependencies:
<h4>For backend:</h4>

```
cd app && npm install
```

<h4>For frontend:</h4>

```
cd frontend && npm install
```
